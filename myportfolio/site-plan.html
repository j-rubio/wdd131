Overview

Purpose

This portfolio website introduces me as a software developer with a strong background in electronic engineering and IT, showcasing the technologies, programming languages, and tools I have worked with. It highlights my transition into software development, shares my educational and professional journey, and features a collection of my recent projects.

Audience

The site is intended for potential employers, collaborators, and peers in the tech industry who are interested in my technical skills, past experience, and current capabilities as a developer.

Dynamic elements

I will use JavaScript with ES modules to dynamically generate and inject reusable components—such as the header, footer, avatar, project cards, and tech icons—into my HTML pages. Each component will be defined in its own module for modularity and maintainability, allowing me to import and render them on specific pages like the Home or Technologies sections. Additionally, I'll use utility modules to manage shared data (e.g., arrays of objects for projects or technologies) and helper functions for tasks like loading stylesheets dynamically or generating internal navigation links. This structure helps keep my codebase organized and allows my site content to be easily updated through JavaScript rather than hardcoded HTML.

Wireframes

Home

The wireframes focus on structure and layout, but they leave out important content, interaction, and visual design details. For instance, in the home page the avatar's style will be realistic, and a grading color pallete will reinforce my personal brand. I will use a sticky header that will stay at the top while scrolling. In addition, a hamburger menu appear under 700px to comply with the responsive design. The footer includes actionable links to phone and email. Also, I plan to use a background video if that does not penalize performance.
